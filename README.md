Обход в ширину (Breadth‑First Search, BFS)
Идея. Посещение узлов по уровням: сначала корень, затем все узлы первого уровня слева направо, затем второго и т. д.

Ключевая структура данных. Очередь (FIFO). Это классический пример применения очередей — ты уже разбирал очереди, и здесь они работают ровно по принципу FIFO.

Алгоритм:

Если корень None — завершаем.
Помещаем корень в очередь.
Пока очередь не пуста:
извлекаем узел из начала очереди;
обрабатываем его (например, выводим значение);
добавляем в очередь левого ребёнка (если есть);
добавляем правого ребёнка (если есть).

Пример BFS на Python
python
from collections import deque

class Node:
    def __init__(self, key):
        self.key = key
        self.left = None
        self.right = None

def bfs(root):
    if root is None:
        return []
    q = deque([root])
    result = []
    while q:
        node = q.popleft()
        result.append(node.key)
        if node.left:
            q.append(node.left)
        if node.right:
            q.append(node.right)
    return result
Обход в глубину (Depth‑First Search, DFS) — «в длину»
Это семейство обходов, где алгоритм идёт вглубь одной ветки, прежде чем переходить к другой. Основные виды:

Preorder (прямой): корень → левое поддерево → правое.
Inorder (симметричный): левое → корень → правое (в BST даёт отсортированный порядок).
Postorder (обратный): левое → правое → корень (удобно для удаления дерева, вычисления выражений).
Реализация. Возможна:

рекурсивная — наглядно, хорошо иллюстрирует рекурсию (ты просил добавлять про рекурсию);
итеративная со стеком — чтобы избежать переполнения стека вызовов на глубоких деревьях.

Рекурсивный DFS (все три вида)
python
def dfs_preorder(node, res):
    if node is None:
        return
    res.append(node.key)
    dfs_preorder(node.left, res)
    dfs_preorder(node.right, res)

def dfs_inorder(node, res):
    if node is None:
        return
    dfs_inorder(node.left, res)
    res.append(node.key)
    dfs_inorder(node.right, res)

def dfs_postorder(node, res):
    if node is None:
        return
    dfs_postorder(node.left, res)
    dfs_postorder(node.right, res)
    res.append(node.key)
Итеративный DFS (preorder) со стеком
python
def dfs_preorder_iterative(root):
    if root is None:
        return []
    stack = [root]
    result = []
    while stack:
        node = stack.pop()
        result.append(node.key)
        # Сначала кладём правого, чтобы левого обработать первым
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return result

Повороты (rotations) — базовые операции перестройки дерева без нарушения порядка BST:

правый поворот (rotate right);

Правый поворот на Python
Пусть у узла y есть левый ребёнок x, у x — правый ребёнок T2. После правого поворота x становится родителем y, а T2 — левым ребёнком y.

python
def rotate_right(y):
    x = y.left
    T2 = x.right

    # Поворот
    x.right = y
    y.left = T2

    return x  # новый корень поддерева
